# Snapshot Save

The FLOW CLI provides a command to save the latest finalized protocol state snapshot

`_10

flow snapshot save <output path>`

## Example Usage[‚Äã](#example-usage "Direct link to Example Usage")

`_10

flow snapshot save /tmp/snapshot.json --network testnet`

### Example response[‚Äã](#example-response "Direct link to Example response")

`_10

snapshot saved: /tmp/snapshot.json`

## Arguments[‚Äã](#arguments "Direct link to Arguments")

### Output Path[‚Äã](#output-path "Direct link to Output Path")

* Name: `output path`
* Valid Input: any valid string path

Output path where the protocol snapshot JSON file will be saved.

## Flags[‚Äã](#flags "Direct link to Flags")

### Host[‚Äã](#host "Direct link to Host")

* Flag: `--host`
* Valid inputs: an IP address or hostname.
* Default: `127.0.0.1:3569` (Flow Emulator)

Specify the hostname of the Access API that will be
used to execute the commands.

### Network Key[‚Äã](#network-key "Direct link to Network Key")

* Flag: `--network-key`
* Valid inputs: A valid network public key of the host in hex string format

Specify the network public key of the Access API that will be
used to create a secure GRPC client when executing the command.

### Network[‚Äã](#network "Direct link to Network")

* Flag: `--network`
* Short Flag: `-n`
* Valid inputs: the name of a network defined in the configuration (`flow.json`)
* Default: `emulator`

Specify which network you want the command to use for execution.

### Filter[‚Äã](#filter "Direct link to Filter")

* Flag: `--filter`
* Short Flag: `-x`
* Valid inputs: case-sensitive name of the result property.

Specify any property name from the result you want to return as the only value.

### Output[‚Äã](#output "Direct link to Output")

* Flag: `--output`
* Short Flag: `-o`
* Valid inputs: `json`, `inline`

Specify in which format you want to display the result.

### Version Check[‚Äã](#version-check "Direct link to Version Check")

* Flag: `--skip-version-check`
* Default: `false`

Skip version check during start up to speed up process for slow connections.
